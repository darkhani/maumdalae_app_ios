# 마음달래 (MaumDalae) — iOS

미술치료 기반 심리 케어 앱 MVP입니다. **펫로스 치유(기억의 숲)** 와 **시니어 활력(인생 사진첩)** 두 여정을 SwiftUI로 제공합니다.

## 기술 스택

- **UI:** SwiftUI (iOS 17+)
- **캔버스:** PencilKit (브러시·스무딩)
- **HTP 촬영:** 카메라/앨범 + CoreMotion 수평 가이드
- **저장:** 로컬 Documents + UserDefaults (서버 연동 전)

## 주요 기능 (1단계 MVP)

| 기능 | 설명 |
|------|------|
| 페르소나 선택 | 펫로스 / 시니어 UI·테마 분기 |
| 디지털 캔버스 | 연필·크레파스·수채·마커, 가이드 템플릿 |
| HTP 검사 | 면책 → 촬영 가이드 → 분석(로컬 휴리스틱) → 추천 세션 |
| 갤러리 | 그림·HTP 기록 로컬 보관 |
| 접근성 | 시니어 고대비·큰 글씨, TTS 음성 안내 |
| 안전장치 | 면책 문구, 위기 시 1393 연결 |

## 실행 방법

1. `MaumDalae.xcodeproj`를 Xcode에서 엽니다.
2. **Signing & Capabilities**에서 본인 Team을 선택합니다.
3. iPhone 시뮬레이터 또는 실기기에서 Run (⌘R).

> HTP 촬영은 **실기기 카메라**에서 가장 정확합니다. 시뮬레이터는 앨범 선택을 사용하세요.

## 프로젝트 구조

```
MaumDalae/
├── App/           # 앱 상태
├── Models/
├── Services/      # HTP 분석(MVP), 로컬 저장
├── Theme/         # 페르소나별 팔레트
└── Views/         # 온보딩, 홈, 캔버스, HTP, 갤러리, 설정
```

## 로드맵 (기획 연계)

- **2단계:** NestJS/Spring API, S3 그림 업로드
- **3단계:** FastAPI + YOLO/SAM + VLM HTP 분석 파이프라인
- **4단계:** 클로즈 베타, 심리 검증

## 면책

HTP 및 감정 분석은 **참고용**이며 전문 진단을 대체하지 않습니다. 앱 내 안내 및 설정의 위기 지원 번호를 확인하세요.
